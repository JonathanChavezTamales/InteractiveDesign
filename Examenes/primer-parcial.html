Examen

Frontend

El frontend es todo lo que vemos como clientes de una página web, sus colores, estilos y su contenido, es una simple página renderizada con html que nos muestra un contenido estructurado y estilizado.

El frontend está construido por diseñadores web que utilizan HTML, CSS y JS para darle estilo contexto a sus creaciones.

Backend

El backend es todo lo que hay detrás, es decir, la lógica de la página, las sesiones, caché, archivos, registros, etc...

El backend puede ser muy complejo o muy simple, este siempre va en un servidor que puede estar en cualquier parte del mundo.

Para el backend se utiliza un servidor que corre un lenguaje de programación que se encargará de manejar todas las peticiones HTTP y dar respuesta a ellas.

CLIENT


El cliente es el dispositivo que accede a nuestras páginas web y el cual puede usarlas con HTML.

CDN


Es un distribuidor de archivos estáticos, como js, css, imágenes, etc... se usa para que al tener usuarios en otras partes del mundo, puedan acceder a ellos más rápido.

Historia de HTML

  • Fue creado en 1995 para compartir documentos científicos.
  • La W3C lo lanza en 1997 de nuevo como un estándar.
  • En 1999 sale una revisión que tiene lo mejor de HTML.
  • Hoy HTML5 es el estándar para páginas web.
